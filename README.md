# <span style="font-size: 32px; font-weight: 700;">Next.js & Tailwind CSS Assignment</span>

## <span style="font-size: 24px; font-weight: 600;">Description</span>
<span style="font-size: 16px;">
A simple assignment project built with Next.js for the frontend framework and styled using Tailwind CSS. This project demonstrates responsive design, modular components, and Next.js routing with a mobile-first, performance-optimized approach.
</span>

## <span style="font-size: 24px; font-weight: 600;">Features</span>
<ul>
<li><span style="font-size: 16px;">Next.js with server-side rendering and static generation</span></li>
<li><span style="font-size: 16px;">Tailwind CSS for fast, responsive styling</span></li>
<li><span style="font-size: 16px;">Clean and modular component structure</span></li>
<li><span style="font-size: 16px;">Mobile-first fully responsive layout</span></li>
<li><span style="font-size: 16px;">Easy navigation with Next.js routing</span></li>
<li><span style="font-size: 16px;">Deployed on Vercel for fast hosting and continuous deployment</span></li>
</ul>

## <span style="font-size: 24px; font-weight: 600;">Installation</span>
<pre style="font-size: 16px;"><code>
1. Clone the repo  
   git clone &lt;your-repo-url&gt;
2. Install dependencies  
   npm install
3. Run development server  
   npm run dev
4. Open http://localhost:3000 to view the app
</code></pre>

## <span style="font-size: 24px; font-weight: 600;">Deployment</span>
<span style="font-size: 16px;">
This app is deployed on Vercel. Push your code to GitHub and connect the repo to Vercel for automatic deployment.
</span>
